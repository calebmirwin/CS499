Detailed Description

A simple course planning system for ABCU.

Author
Caleb Irwin
Date
03/30/2025

This program loads, displays, and searches course information using an AVL-balanced binary search tree. It supports CSV file input, alphabetical traversal, duplicate detection, and structured exception handling.

CSV lines should be structured: <courseId>,<courseName>,<prerequisite1>,<prerequisite2>,...
